To use this app, you have to start the python script called "Viewer.py" and type in the name
of the image file you want to open. To add a new file beside the examples, you have to create
a new .simpix file and structure it as shown in the documentation file "SimpixFormat.txt". Make
sure that the new file is in the same directory as the script and you havent misspelled anything
because this app offers 0 error handling. Thanks for using and make sure to give me some positive
feedback in github
